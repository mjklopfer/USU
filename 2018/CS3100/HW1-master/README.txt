Implement a program that computes the Fibonacci of a specified number, estimates the value of 'e' using the specified number of iterations
(of a Taylor series), or computes the value of 'pi' to the specified number of decimal places.

Valid ranges for the Fibonacci [n] parameter are: 0 to 40
Valid ranges for the 'e' [n] parameter are: 1 to 30
Valid ranges for the 'pi' [n] paramter are: 1 to 10

Only handle a single computation at a time


In the case of no or invalid paramters, the program should show help instructions that looks like:

---Assign 1 Help ---
-fib [n] : Compute the fibonacci of [n]
-e [n] : compute the value of 'e' using [n] iterations
-pi [n] : Compute Pi to [n] digits


Example command line usages of the program:

./Assign1 -fib 6
./Assign1 -e 24
./Assign1 -pi 6
